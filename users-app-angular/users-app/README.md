# О приложении

Это небольшое Angular-приложение для просмотра и управления списком пользователей. Проект работает с внешним REST API [JSONPlaceholder](https://jsonplaceholder.typicode.com/) и показывает базовый CRUD-сценарий: получение списка пользователей, просмотр деталей, создание, редактирование и удаление.

Интерфейс построен на Angular standalone components и NG-ZORRO. Для форм используется Reactive Forms, для запросов к API — Angular HTTP Client.

## Возможности

- просмотр списка пользователей;
- поиск пользователей по имени или email;
- просмотр подробной информации о пользователе;
- создание нового пользователя;
- редактирование существующего пользователя;
- удаление пользователя с подтверждением;
- пагинация таблицы;
- валидация формы пользователя;
- маршрутизация между страницами приложения;
- сообщения об успешных действиях и ошибках.

## Стек технологий

- Angular 20;
- TypeScript;
- SCSS;
- Angular Router;
- Angular HTTP Client;
- Reactive Forms;
- NG-ZORRO Ant Design;
- RxJS;
- JSONPlaceholder API.

## API

Приложение использует внешний сервис:

```text
https://jsonplaceholder.typicode.com/users
```


### Основные части

- `UserService` — сервис для работы с API пользователей.
- `UserListComponent` — страница со списком пользователей, поиском, пагинацией и действиями.
- `UserDetailComponent` — страница подробной информации о пользователе.
- `UserFormComponent` — форма создания и редактирования пользователя.
- `app.routes.ts` — описание маршрутов приложения.
- `app.config.ts` — настройка провайдеров Angular, HTTP Client, Router, анимаций и локализации NG-ZORRO.

## Маршруты

```text
/users              список пользователей
/users/new          создание пользователя
/users/:id          подробная информация о пользователе
/users/:id/edit     редактирование пользователя
```

При открытии корневого адреса приложение автоматически перенаправляет на `/users`.

## Установка и запуск

Перед запуском нужно установить зависимости:

```bash
npm install
```

Запуск проекта в режиме разработки:

```bash
npm start
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:4200
```

Если порт `4200` уже занят, можно запустить приложение на другом порту:

```bash
ng serve --port 4300
```

Пример команды для первого запуска
```
cd E:\Test-Task-for-Yadro\users-app-angular\users-app
npm install
npm start
```

### Поиск

На странице списка пользователей реализована фильтрация по двум полям:

- `name`;
- `email`.

Поиск выполняется на клиентской стороне по уже загруженному списку пользователей.

### Создание и редактирование

Форма пользователя использует Reactive Forms. Обязательные поля:

- имя;
- email.

Для email используется стандартная проверка корректности формата.

### Удаление

Перед удалением пользователя показывается подтверждение. После успешного ответа API пользователь удаляется из локального списка на странице.

## Примечание по сохранению данных

Так как проект работает с JSONPlaceholder, реальные данные на сервере не изменяются. Это значит, что приложение корректно отправляет запросы на создание, редактирование и удаление, но сервер не хранит эти изменения постоянно.
